MySQL JDBC Driver Required

To run this project, you need to download the MySQL JDBC driver:

1. Go to: https://dev.mysql.com/downloads/connector/j/
2. Download the latest version (8.0.x)
3. Place the .jar file in this lib folder
4. Rename it to: mysql-connector-java.jar

Or download directly from Maven:
https://repo1.maven.org/maven2/mysql/mysql-connector-java/8.0.33/mysql-connector-java-8.0.33.jar

The project expects the driver to be named: mysql-connector-java.jar
